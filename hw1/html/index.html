Андрей Пирожков

У компании в которой я работаю нацеленность в основном на дизайн системы. Имеются несколько не сложных проектов которые мы поддерживаем по части кода и для этого нам требуется только dev среда. Проекты все простые и по большей части базируются на WordPress. Исходя из этого мы используем предлагаемые местным провайдером netangel "Облачные VDS" на базе Ubuntu 18.04 с характеристиками 2CPU 2Gb RAM 70GB SSD. Нам доступны все настройки системы. Это нам обходится ~1500р/мес.

В сравнение ставлю оценку свобки Amazon EC2 с темиже характеристиками для 1-го экземпляра. Ежемесячная стоимость составит ~ 1400р.

Итог. Облачное решение дешевле, но не назначительную сумму. Отсутствует какая либо потребность в DevOps специалисте, а также Production и Stage инфраструктур.
