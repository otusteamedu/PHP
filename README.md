# Запуск

1. Устанавливаем пакет
    ```bash
    composer install
    ```

2. Запускаем с помощью PHP CLI

    ```bash
    php app.php
    ```

    Должно вывести: `Hello from https://github.com/petrov-aa/otus-hw4-2`
