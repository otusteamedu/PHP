#### Первый шаг
######Перейти в директорию **code** и выполнить команду
```
composer install
```
#### Второй шаг - Запуск проекта, вернуться в корень проекта, выполнить команду
```
docker compose up
```
#### Третий шаг
######Перейти в директорию **code** выполнить команды в 2 терминалах/консолях
```
php app.php server
php app.php client
```
